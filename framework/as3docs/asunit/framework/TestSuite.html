Classpublic class TestSuite
InheritanceTestSuite TestCase Assert flash.events.EventDispatcher
ImplementsTest

A TestSuite is a Composite of Tests. It runs a collection of test cases. Here is an example using the dynamic test definition. 
  TestSuite suite = new TestSuite();
  suite.addTest(new MathTest());
  suite.addTest(new OtherTest());
